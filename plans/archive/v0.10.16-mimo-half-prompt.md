## Failure modes (NAKE escalation triggers)

Either of the following triggers `M-1 NAKE` + cycle pause (parent will surface to maintainer):

- **D2/D3 fix introduces a new failure** (net test count decreases vs pre-D2/D3) — likely an unexpected coupling.
- **A fix requires >50 LoC of production-code changes** (signals the D1 classification was wrong; needs more diagnostic work first).
- **The fix would undo a v0.10.15 hygiene contract** (test must observe post-v0.10.15 behavior, NEVER pre-v0.10.15).
- **Test runtime increases by >2× vs pre-fix** (likely the fix is a bare `sleep` rather than a real fix).
- **D1 over-defers** - if `>4 of 9` failures classify into bin (e) `deferred`, the cycle's success criterion (all 9 GREEN) is at risk. Post `M-1 NAKE D1: <count>/9 classified as deferred > 4 - cycle scope needs re-evaluation` immediately so the parent can re-scope before D2 starts.
