### Considered and rejected

- **Test suite requiring Docker with no hermetic fallback** (finding #11): Adding SQLite fallback for 353 pytest tests is L/XL effort (SQLite ≠ Postgres on JSONB, CHECK constraints, array types). The project's test pattern is "Docker infra + fast hermetic tests." Documented in the existing pattern; not worth the SQLite compatibility maintenance tax.
- **Rust + PyO3 parser binding** (finding DIR-03 from subagents): Would require benchmarking first. Deferred until a benchmark plan is prioritized by the maintainer.
